# Nexcent Landing Page (React Learning Lab) 🚀

This repository is dedicated entirely to mastering **React** and **TypeScript** fundamentals. The goal of this project is to take a static corporate Figma design and break it down into interactive, component-driven architecture by hand, rather than relying on automatic code generators.

## 🎯 Learning Objectives Conquered
* **Thinking in Components:** Transitioning from long, static HTML documents to reusable UI lego bricks.
* **Dynamic Data Rendering:** Utilizing JavaScript arrays and `.map()` loops to render layout elements dynamically instead of hardcoding text tags.
* **Layout Mechanics:** Implementing strict semantic HTML structure and CSS Flexbox to prevent layout overlap without resorting to messy positioning hacks.
